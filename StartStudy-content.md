## Content Strategy Notes

### Tone & Voice
- **Professional yet approachable:** Academic credibility with warm, parent-friendly language
- **Clear and concise:** Short descriptions that quickly communicate value
- **Action-oriented:** Links use verbs (Browse, Discover, Take, See, View)

### Key Messaging Principles
1. **Age-appropriate labeling:** Every program includes age ranges and grade equivalents
2. **Value propositions:** Each description highlights the unique benefit/focus
3. **Competitive differentiation:** "Get Ready" emphasized as "World's only" and "92% pass rate"
4. **Educational terminology:** Uses recognized terms (O-Level, IGCSE, FSc, Cambridge)
5. **Quick decision-making:** Users can identify their path in seconds

### Differentiation Hierarchy
1. **Primary highlight:** Get Ready Courses (featured twice, unique badge, statistics)
2. **Standard programs:** All other grade levels (equal visual weight)
3. **Subjects:** Clear, simple categorization
4. **Quick actions:** Easy access to trial/exploratory actions

### Content Updates & Maintenance
- **Pass rate statistics:** Update annually based on latest data
- **Age ranges:** Review annually to ensure accuracy with grade-level standards
- **Program names:** Keep consistent across all site areas
- **URLs:** Maintain stable URL structure; use redirects if changes needed

### Link Text Clarity
- All links have descriptive text (no "click here" or "learn more")
- Icon descriptions provided for visual context
- Consistent naming across menu and destination pages
